The Verification Engine (Observer Side)Pelihat berita melakukan audit otomatis melalui browser mereka:CID Resolution: Browser mengambil metadata dari Storacha via IPFS Gateway.Proof Verification: Sistem mengecek nullifier_hash di World Chain untuk memastikan pengunggah belum melakukan spamming.Integrity Check: Browser menghitung ulang hash dari file yang ditampilkan. Jika ComputedHash === ContentHash (yang ada di dalam metadata), maka indikator "TRUE HUMAN WITNESS" menyala hijau.3.
